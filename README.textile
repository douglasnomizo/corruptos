h1. Corruptos

Aplicação desenvolvida como projeto para disciplinas de Processo e Desenvolvimento de Sistemas III e Gerência de Projetos III da Pontifícia Universidade Católica de Goiás.

h2. Ruby on Rails

Esta aplicação utiliza:

* Ruby versão 1.9.3
* Rails versão 3.2.8
* Outras bibliotecas listadas no Gemfile

h2. Ferramentas

* Template Engine: ERB
* Front-end Framework: Twitter Bootstrap (Sass)
* Form Builder: SimpleForm
* Authentication: Devise
* Authorization: CanCan
